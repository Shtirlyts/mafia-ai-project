# AI Мафия - Веб-игра с AI агентами

Веб-версия классической игры "Мафия" с поддержкой трёх режимов: только AI-агенты, смешанный (люди + агенты), только люди.

## 🎮 Особенности

- **Три режима игры**: только AI, смешанный режим, только люди
- **Реалистичные AI-агенты**: неотличимы от людей в чате
- **Множественные роли**: Мафия, Комиссар, Доктор, Мирные жители
- **Тест Тьюринга**: статистика распознавания AI после игры
- **Responsive дизайн**: адаптивный интерфейс

## 🛠 Технологии

- **React 18** с TypeScript
- **Vite** - сборщик проекта
- **Tailwind CSS v4** - стилизация
- **Lucide React** - иконки
- **TypeScript** - строгая типизация

## 🎯 Компоненты

### LobbyScreen
Экран создания и ожидания игроков:
- Код комнаты для приглашения
- Список подключенных игроков (люди/AI)
- Настройки игры
- Автозаполнение AI-агентами

### GameScreen
Основной игровой процесс:
- Фазы день/ночь с таймером
- Чат для дискуссий
- Система голосования
- Ночные действия для ролей
- Список живых и выбывших игроков

### ResultsScreen
Результаты игры:
- Раскрытие ролей
- Показ, кто был AI, а кто человеком
- Статистика "Теста Тьюринга"
- Метрики распознавания AI

### SettingsModal
Настройки игры:
- Выбор режима игры
- Количество игроков
- Длительность фаз
- Распределение ролей

## 🚀 Запуск проекта

Проект уже настроен и работает в среде Figma Make с Vite.

Для локальной разработки:

```bash
# Установка зависимостей
pnpm install

# Запуск dev-сервера
pnpm dev

# Сборка проекта
pnpm build
```
